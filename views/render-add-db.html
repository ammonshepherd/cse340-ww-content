Connect the Database Service

In a browser go to Render.com and sign-in using your GitHub credentials.

We can now connect the database to the web application.

  1. While on the dashboard for the web application server, click the "Environment" item on the left side of the screen.
  2. Click the "Add Environment Variable" button.
  3. Click in the "Key" text box and type "DATABASE_URL".

    Also add this key to the .env file within Visual Studio Code.

  4. Create the DATABASE URL using the information from the DATABASE LOGIN INFO file and type it into the value field. It should be in the format

    postgresql://username:password@127.0.0.0:5432/DatabaseName

    NOTE that the DatabaseName and username are actually the same. Your password is just the numbers of your student ID.

    Also add this value to the .env file.

    Your .env file should now have a line like this:

    DATABASE_URL='postgresql://username:password@127.0.0.1:5432/DatabaseName'
  5. Click the "Save Changes" button.
